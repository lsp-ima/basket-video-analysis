# 動画分析 ｜ TEAM BOARD (β)

バスケットボールの試合動画を分析するWebアプリ。ブラウザだけで動作し、**動画はサーバーに一切アップロードされません**(端末内で処理)。

## 機能

- 📋 **スタッツ記録** — 動画(ファイル / YouTube URL)を再生しながらワンタップで記録: 得点・リバウンド・アシスト・スティール・ターンオーバー。個人を特定できないプレーはチーム単位で記録可能
- 🤖 **AI自動計測** — 選手を自動検出・追跡してプレータイム・走行距離を算出(動画ファイルのみ)
- 🎯 **ターゲット追跡** — ユニフォーム色を指定して特定チームの断片を抽出、切り出し画像を見て背番号を割り当てて名寄せ
- 📚 **記録** — 動画ごと・日付ごとに端末内へ保存、CSV出力

## 推奨撮影条件(AI計測)

- カメラ**固定**(三脚・手持ち/ズーム/パン不可)
- コート全体が入る高めの位置(2階ギャラリー等)
- 1080p以上(背番号の識別を重視するなら4K)

## 技術・ライセンス

- 選手検出: [Ultralytics YOLOv8n](https://github.com/ultralytics/ultralytics)(AGPL-3.0)から変換したONNXモデルを同梱
- 推論: [ONNX Runtime Web](https://onnxruntime.ai/)(WebGPU / WASM)
- 上記に伴い、本リポジトリのソースコードは **AGPL-3.0** で公開しています

走行距離・プレータイムは映像からの推定値であり、目安としてご利用ください。
